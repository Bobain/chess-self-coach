# Samir, le grinder discipliné

**Elo** : ~1100-1400
**Archetype** : Grinder — values efficiency and routine

## Profile

Joue 2-3 parties par jour, s'entraîne régulièrement. Comprend les tactiques de base, commence à saisir les idées positionnelles. Lit la notation algébrique sans difficulté. Familier avec les outils d'analyse (Lichess analysis board, chess.com game review).

## Goal

Répétition espacée efficace. Veut passer rapidement sur les positions connues et s'attarder seulement sur celles qu'il rate. Suit sa progression sur la durée.

## Observed behavior

- Fait ses sessions training comme une routine quotidienne (5-10 min)
- Connaît la plupart de ses erreurs récurrentes et les reconnaît
- Veut le minimum de friction entre les positions : réponse → feedback → next
- Utilise le game review pour ses parties récentes, compare avec chess.com
- Apprécie les stats (accuracy, nombre de positions vues, taux de réussite)

## Frustrations with current UI

- Pas de stats de session ni de progression sur la durée (combien de positions maîtrisées ?)
- Le game review est bon mais les classifications ne sont pas toujours alignées avec chess.com
- Le menu hamburger a trop d'items — il n'utilise que Training et Analysis
- Pas de raccourcis clavier dans le training (Espace pour next, Entrée pour retry)

## What would help

- Dashboard de progression (positions apprises, taux par catégorie)
- Raccourcis clavier pour le training flow (vitesse)
- Filtre par type d'erreur (blunders seulement, ou inaccuracies seulement)
- Notification quand de nouvelles positions sont disponibles après analyse

## Usage pattern

Sessions courtes (5-10 min) matin et soir, desktop et mobile à parts égales. Valorise l'efficacité du flux plus que la profondeur pédagogique.
