# Nadia, l'analyste post-partie

**Elo** : ~1000-1500
**Archetype** : Reviewer — deep analysis after each session

## Profile

Joue principalement des parties longues (rapid 15+10). Après chaque session de jeu, passe du temps à revoir ses parties. Utilise chess.com Game Review et l'analysis board de Lichess comme références. S'intéresse aux ouvertures et veut comprendre où elle sort de la théorie.

## Goal

Comprendre ses parties en profondeur. Voir quand elle a quitté la théorie d'ouverture, identifier les moments critiques, explorer des variantes alternatives.

## Observed behavior

- Passe 15-30 min par session de review, parcourt chaque coup
- Compare l'eval bar et le score chart avec ce qu'elle voit sur chess.com
- S'attarde sur les moments où l'avantage bascule (turning points)
- Cherche les patterns dans ses erreurs (toujours les mêmes types de positions ?)
- Veut pouvoir partager une position intéressante avec un ami

## Frustrations with current UI

- Pas d'exploration de variantes dans le game review (ne peut pas jouer "et si ?")
- La PV est tronquée à 8 coups — parfois elle veut voir la suite complète
- Pas de moyen de marquer/bookmarker un moment clé dans une partie
- Le score chart est non-interactif au-delà du clic pour naviguer
- Manque de contexte d'ouverture détaillé (juste le nom ECO, pas les idées)

## What would help

- Mode exploration dans le game review (jouer des coups alternatifs, voir l'eval)
- Annotations ou bookmarks sur les moments clés
- Comparaison côte à côte de 2 parties sur la même ouverture
- Plus de détails sur le departure from theory (quelle était la ligne théorique ?)
- Export/partage d'une position vers Lichess analysis

## Usage pattern

Desktop principalement, sessions longues après les parties. Utilise aussi le training mais c'est secondaire pour elle.
